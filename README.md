# Global Survey Overview: Demographics & Tech Stack Trends

An executive-style **Power BI dashboard** that explores global developer survey data to uncover insights on **respondent demographics**, **top technology stacks**, and **salary trends by experience**.

## Project Summary

This dashboard provides an interactive overview of:
- Global respondent distribution (map)
- Demographics: gender & age
- Top technology stacks (languages, frameworks, databases, platforms)
- Average salary trends based on years of experience

---

## Objectives

- Deliver a clean, executive-style dashboard for quick decision-making
- Identify dominant respondent demographics and geographic distribution
- Compare the most popular technologies used globally
- Analyze salary trends across experience levels
- Enable country-level drill-down via interactive filtering

## Dashboard Sections

### 1) Overview & Demographics
- Respondents by **Country** (Map)
- Respondents by **Gender**
- Respondents by **Age**

### 2) Top Tech Stack
- **Top Languages:** JavaScript, HTML, CSS, SQL, Java  
- **Top Frameworks:** Node.js, Angular, React, .NET Core, Spring  
- **Top Databases:** MySQL, SQL Server, PostgreSQL, MongoDB, SQLite  
- **Top Platforms:** Linux, Windows, Android, AWS, Mac OS  

### 3) Salary Trends
- **Avg Salary Trends by Experience (USD)**
- Trend shows salary growth along years of experience (career progression)

## Data Analysis Workflow

### 1) Data Preparation & Cleaning (Microsoft Excel)
Tasks performed in Excel:
- Removing duplicates and handling missing values
- Standardizing categorical values (e.g., gender, country)
- Formatting and validating numeric fields (e.g., salary)
- Creating clean structured tables for analysis-ready dataset (Power Query)

### 2) Data Modeling & Reporting (Power BI)
- Data model setup and relationships
- KPI calculations using DAX
- Interactive visuals and slicers
- Executive dashboard layout & formatting
